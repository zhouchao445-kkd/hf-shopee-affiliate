# 🎯 Shopee 精选联盟数据分析

Shopee 精选联盟（Affiliate）数据分析工具。

## 功能
- 📊 数据概览：订单总数、佣金统计、达人数量
- 📱 平台对比：各推广平台订单/佣金分布
- 🏆 达人排行：TOP 20 高佣金达人
- 📈 每日趋势：每日订单和佣金变化
- 🎁 促销类型：不同促销活动的佣金对比

## 使用方法
1. 从 Shopee 精选联盟后台导出「达人转化报告」
2. 上传 .xlsx / .xls / .csv 文件
3. 点击「开始分析」
